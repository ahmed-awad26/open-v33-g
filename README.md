# OpenContactsV2

OpenContactsV2 is a vault-first Android starter for a privacy-focused private contacts application.

## Included in this revision
- Multi-module Android project
- Compose + Material 3 UI shell
- Hilt dependency injection
- Room + SQLCipher-ready per-vault databases
- Vault registry database
- Vault session manager
- Global PIN app lock foundation
- Biometric unlock prompt wiring
- Security settings screen
- Create / edit / delete contacts
- Create / rename / delete vaults

## Notes
This is still a starter implementation, not the full end-product.
Cloud backup, full per-vault security policies, call-log integration, reminders, and import/export flows still need production hardening.
